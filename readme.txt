== Description ==

This plugin integrates wordpress and Trello.
When you post an article to wordpress, this plugin will create a card in Trello.

== Installation ==
1. Upload All files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 「Trello連携」 page under Settings menu.
4. Enter the API key you got in "Trello Key"
5. Enter the token you got in "Trello Token"

== Frequently Asked Questions ==
= Where can I get the Trello developer API key? =

After you log in to Trello, go to the following URL.
[TrelloAppKeyGenerate](https://trello.com/1/appKey/generate/ "Your favorite software")
